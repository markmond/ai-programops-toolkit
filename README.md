# ai-programops-toolkit
Open-source operating artifacts for institutional AI program execution.


AI ProgramOps Toolkit helps Chief AI Officer offices, Directors of AI Programs, AI task forces, and institutional leaders turn AI strategy into governed, budgeted, measurable, stakeholder-aligned execution.

## What this is

A practical library of:

- Templates
- Workflows
- Stage gates
- Dashboards
- Checklists
- Briefs
- Rubrics
- Reporting artifacts
- Filled examples

## Who this is for

- Chief AI Officer offices
- Directors of AI Programs
- Institutional AI task forces
- Higher education leaders
- Healthcare innovation offices
- Research administrators
- Public-sector AI teams
- Consultants supporting institutional AI adoption

## What this is not

This is not:

- A model-development repository
- A prompt library
- A legal advice resource
- A policy-only toolkit
- An OpenAI-affiliated project

## First 90 Days Kit

The first release focuses on the artifacts an institutional AI program office needs in its first 90 days:

- AI Initiative Inventory
- AI Program Portfolio Tracker
- AI Initiative Intake Form
- AI Program Charter Template
- AI Prioritization Matrix
- AI Risk Classification Framework
- Governance Routing Map
- Responsible AI Review Checklist
- RAID Log
- Milestone Tracker
- CAIO Weekly Brief
- AI Program Budget Template
- AI Metrics Dictionary
- Pilot Evaluation Report
- Stakeholder Map
- Stakeholder Engagement Plan

## Project status

Sprint 1: Public Skeleton

This repository is currently establishing the public structure, documentation site, contribution system, and First 90 Days Kit map.
